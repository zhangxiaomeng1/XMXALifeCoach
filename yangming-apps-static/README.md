# Yangming Apps Static Homepage

Open `index.html` in a browser to preview the static site.

## Files
- `index.html`: single-page homepage
- `style.css`: responsive Apple-style CSS
- `assets/icons/`: app icons cropped from the provided screenshot
- `assets/homepage-og.png`: Open Graph preview image
- `privacy.html`, `terms.html`: placeholder legal pages

## Replace before production
- Replace cropped icons with original 1024x1024 App Store icons
- Replace placeholder privacy/terms pages
- Confirm contact email and domain
- Add final screenshots or device mockups
